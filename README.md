## Advantage 框架的优势
* 1.文件少，代码简洁
* 2.不依赖任何其他第三方库
* 3.具备较高自定义性

## Requirements 要求
* iOS 7+
* Xcode 8+


## Principle 实现原理
在TableView的Header上添加Button，通过一个可变字典记录展开、收缩状态。
